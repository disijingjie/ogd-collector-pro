## 二、通用表格设计原则

### 2.1 字体与排版

| 元素 | 规范 |
|:---|:---|
| 表头字体 | 黑体/宋体加粗，10.5pt（五号） |
| 数据字体 | 宋体，10.5pt（五号） |
| 注释字体 | 宋体，9pt（小五号），灰色 |
| 行高 | 最小1.2倍行距，长表格1.5倍 |
| 表格宽度 | 通栏（页宽）或半栏，避免超宽 |

### 2.2 对齐规范

| 数据类型 | 对齐方式 | 示例 |
|:---|:---|:---|
| 文本/省份名称 | 左对齐 | 广东、浙江 |
| 数字/得分 | 右对齐 | 0.791、1.000 |
| 百分比 | 右对齐，保留1位小数 | 94.2% |
| 排名序号 | 居中对齐 | 1、2、3 |
| 分类标签 | 居中对齐 | 标杆型、DEA有效 |

### 2.3 视觉层次（适合黑白打印）

```
┌─────────────────────────────────────────────────────────┐
│  表X-X 表格标题（黑体加粗，置于表格上方居中）              │
├─────────────────────────────────────────────────────────┤
│  表头行：浅灰底纹（15%灰度）+ 黑色下框线（1.5pt）        │
│  ─────────────────────────────────────────────────────  │
│  数据行1：白色底纹                                        │
│  数据行2：浅灰底纹（5%灰度）← 斑马纹，便于横向阅读        │
│  数据行3：白色底纹                                        │
│  数据行4：浅灰底纹（5%灰度）                              │
├─────────────────────────────────────────────────────────┤
│  关键行：粗体标记（如TOP3、DEA有效行）                    │
├─────────────────────────────────────────────────────────┤
│  注：数据来源说明（小五号，灰色，置于表格下方）            │
└─────────────────────────────────────────────────────────┘
```

### 2.4 框线设计（打印友好）

- **三线表**（推荐用于数据表格）：仅保留顶线、表头下线、底线，共三条横线
- **全框线**（推荐用于复杂矩阵）：细外框线(0.5pt) + 表头下粗线(1.5pt)
- **避免**：竖向框线（影响阅读流畅性）、双线、阴影效果

## 三、重点表格优化方案

- 30行数据过长，单页无法完整呈现

## 五、Word排版操作建议

### 5.1 表格转Word流程

1. **Markdown → HTML**：使用 Pandoc 或 Python 脚本转换
2. **HTML → Word**：直接粘贴到Word，或另存为.docx
3. **样式调整**：
   - 全选表格 → 设计 → 表格样式 → 新建表格样式
   - 设置"网格型"为基础，删除竖向框线
   - 表头行设置"底纹：灰色-15%"
   - 数据行设置"交替行：灰色-5%"

### 5.2 长表格分页处理

| 场景 | 处理方式 |
|:---|:---|
| 表格跨页 | 表头行设置"重复标题行" |
| 30行以上大表 | 拆分为上下两个子表，或横向排版 |
| 7列以上宽表 | 拆分为两个关联表，或缩小字号至9pt |
| 附录表格 | 可使用更小字号（9pt），行高压缩 |

### 5.3 推荐Word表格样式参数

| 参数 | 设置值 |
|:---|:---|
| 表格样式 | 三线表或自定义网格 |
| 外框线 | 0.5pt 黑色 |
| 表头下框线 | 1.5pt 黑色 |
| 内部横线 | 0.25pt 浅灰（可选） |
| 竖框线 | 无 |
| 表头底纹 | 灰色-15% |
| 交替行底纹 | 灰色-5% |
| 单元格边距 | 上3pt 下3pt 左5pt 右5pt |
