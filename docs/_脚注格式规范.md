# 武汉大学博士论文脚注引用格式规范（GB/T 7714-2015）

## 格式特征（基于模板PDF分析）

### 1. 脚注编号
- 使用带圈数字：① ② ③ ④ ⑤ ... 
- 标注在正文引用处的右上角
- 每页单独编号，从①开始

### 2. 脚注位置
- 放在每页底部（页脚区域）
- 与正文之间用短横线分隔

### 3. 文献类型标识
- [J] 期刊论文（Journal）
- [D] 学位论文（Dissertation）
- [M] 专著（Monograph）
- [C] 会议论文（Conference）
- [R] 报告（Report）
- [EB/OL] 网络文献（Electronic Bulletin/Online）
- [N] 报纸（Newspaper）
- [S] 标准（Standard）

### 4. 格式示例

**中文期刊论文：**
```
作者.题名[J].刊名,年,卷(期):页码.
```
例：田宏. 全球研究理事会（GRC）2014年全体大会在北京召开[J]. 中国科技期刊研究, 2014, 25(6):862.

**英文期刊论文：**
```
作者.题名[J].刊名,年,卷(期):页码.
```
例：Nature Methods Editorial Board. Reviewing computational methods[J]. Nature Methods, 2015, 12(12):1099.

**学位论文：**
```
作者.题名[D].学校,年.
```
例：常静. 百度百科及开源社区参与者的动机与行为关系的实证研究[D]. 华南理工大学, 2010.

**专著：**
```
作者.书名[M].出版地:出版社,年:页码.
```

**网络文献：**
```
作者.题名[EB/OL].(发布日期)[引用日期].URL.
```

**多作者（3人以内）：**
```
作者1, 作者2, 作者3.题名[J].刊名,年,卷(期):页码.
```

**多作者（4人及以上）：**
```
作者1, 作者2, 作者3, et al.题名[J].刊名,年,卷(期):页码.
```

### 5. 引用位置规则
- 直接引用他人观点：在句末标点之前标注
- 引用数据或事实：在数据或事实之后标注
- 引用整段内容：在段落末尾标注
- 多次引用同一文献：每次引用都需标注脚注（不能省略）

### 6. 注意事项
- 作者姓名之间用逗号分隔
- 英文作者姓在前名在后（或按原文格式）
- 期刊名用斜体（Word中）或正常字体（Markdown中）
- 页码用冒号分隔，不加"页"字
- 年份后不加"年"字（但中文文献有时保留）
- 脚注内容左对齐，字号小于正文

## 本论文的特殊处理

由于本论文使用Markdown格式，脚注处理方式：
1. Markdown中使用 `[^1]`, `[^2]` 等格式标记脚注
2. 在文档末尾或每节末尾集中列出脚注内容
3. 最终通过pandoc转换为Word时，会自动处理为页面底部脚注

## 参考文献列表格式

在论文末尾的"参考文献"章节，使用顺序编码制：
```
[1] 作者.题名[J].刊名,年,卷(期):页码.
[2] 作者.题名[D].学校,年.
```

与脚注的区别：
- 脚注：每页底部，带圈数字，引用具体页码
- 参考文献：论文末尾，方括号数字，不标注具体页码
