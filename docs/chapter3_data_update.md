# 第三章 数据引用更新说明

## 需要更新的关键数据点

### 3.3.2 4E框架在政府数据开放领域的适配

**新增实证支撑：**

> "本研究基于22个省级平台的实证数据，对4E框架的四个维度进行了操作化测量。供给保障维度以标准化数据集数量衡量，平台服务维度以功能完备性评分衡量，数据质量维度以结构化率和元数据完整性衡量，利用效果维度以应用案例数量衡量。评估结果显示，四个维度的变异系数分别为1.79（供给）、0.45（平台）、0.38（质量）、0.62（效果），供给维度的异质性最大，表明各平台在数据开放规模上差异最为显著。"

### 3.3.3 指标权重确定

**新增AHP-熵权组合赋权结果：**

> "本研究采用AHP-熵权组合赋权法确定指标权重。AHP法基于专家判断，供给保障权重0.25、平台服务权重0.20、数据质量权重0.20、利用效果权重0.30、公平性权重0.05；熵权法基于数据变异程度，供给保障权重0.35、平台服务权重0.15、数据质量权重0.15、利用效果权重0.30、公平性权重0.05。组合赋权结果为：供给保障0.30、平台服务0.175、数据质量0.175、利用效果0.30、公平性0.05。"
