### 2.3 采集系统升级路线（V3 → V3.5）

**当前V3功能**：
- 22省数据采集结果展示
- 基础统计（成功率、数据集数量）
- 规则映射表

**V3.5升级目标**：
1. **采集过程可追溯**：每个平台的采集日志、截图、HTML快照
2. **数据口径标准化自动计算**：输入原始数量，自动输出标准化数量
3. **8省替代渠道采集**：政务服务网、数据局官网等替代形式的采集模块
4. **平台转型监控**：定期检测平台状态变化（维护/转型/关闭）
5. **论文数据一键导出**：按章节导出所需的表格、图表、数据
