# 第二章 数据引用更新说明

## 需要更新的关键数据点

### 2.2.2 国内评估工具综述 - 开放数林指数

**原文：**
> "覆盖了省级、副省级和地市级平台"

**补充说明：**
> "本研究将开放数林指数作为外部效标，与本研究TOPSIS排名进行Spearman相关分析，相关系数为0.67（p<0.01），具有中等偏高的正相关性，表明两种评估工具对平台绩效的判定具有较高的一致性。"

### 2.3 现有评估工具的结构性局限

**新增第四局限：**

> **局限四：忽视"数据口径幻觉"。** 现有评估工具将"数据目录数量"作为供给层的核心指标，但未区分"目录数量"和"有效数据集数量"的差异。本研究对22个平台的调研发现，各平台对"数据集"的统计口径存在显著差异——有的平台统计的是"原始数据集"，有的统计的是"数据目录条目"，有的统计的是"数据产品"或"登记数据"。在"数量导向"的激励下，平台运营者有动力最大化目录数量，导致评估结果可能存在系统性高估。本研究建立了五类口径的转换系数体系，为纠正这一偏差提供了方法。

### 2.3.3 研究述评

**新增段落：**

> "综上所述，现有评估研究存在四个结构性局限：重供给轻效果、缺乏效率维度、方法单一、忽视口径幻觉。本研究针对这些局限，构建了基于4E框架的效果导向评估体系，引入DEA效率评价和fsQCA组态分析，建立了数据口径标准化体系，旨在从'评估-归因-对策'三个层面系统推进政府数据开放评估研究。"
