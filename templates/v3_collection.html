七步采集流水线
Step 1: 种子URL发现
基于国务院《政务信息资源共享管理暂行办法》及各省数据局官网，建立初始URL种子库（88个平台入口）
Step 2: 页面类型识别
发送HEAD请求识别页面类型：静态HTML / 动态JS渲染 / REST API接口，自动选择采集策略
Step 3: 目录结构解析
解析平台目录树（主题分类 → 部门分类 → 数据集列表），建立层级映射关系
Step 4: 元数据提取
提取每条数据集的名称、描述、格式、标签、更新日期、下载量、提供部门等核心元数据字段
Step 5: 数据清洗标准化
统一日期格式（ISO 8601）、标准化部门名称（参照GB/T 2260）、格式映射（CSV/XLS/JSON/API）
Step 6: 质量校验入库
完整性校验（必填字段≥80%）、一致性校验（省份-平台关联正确）、重复性校验（URL去重）
Step 7: 增量更新维护
基于哈希值比对实现增量更新，仅采集变更数据，记录每次采集的完整日志
各省平台技术架构差异
静态页面型（约40%）
代表：山东、浙江、广东
技术：传统服务器端渲染，HTML+CSS
采集：Requests + BeautifulSoup直接解析
优势：响应快、结构稳定、易于解析
动态渲染型（约35%）
代表：北京、上海、四川
技术：Vue/React前端框架，异步加载
采集：Selenium + ChromeDriver渲染
优势：交互丰富、用户体验好
接口API型（约20%）
代表：贵州、福建、深圳
技术：RESTful API + JSON数据返回
采集：直接调用API端点获取JSON
优势：数据结构规范、易于批量获取
混合架构型（约5%）
代表：部分中西部市级平台
技术：静态列表+动态详情+文件下载
采集：多策略组合，先静态后动态
挑战：需要定制化解析规则

python # 静态页面采集：山东省数据开放平台 import requests from bs4 import BeautifulSoup class StaticCollector: def __init__(self, platform): self.platform = platform self.session = requests.Session() self.session.headers.update({'User-Agent': 'Mozilla/5.0...'}) def collect(self, url): resp = self.session.get(url, timeout=30) soup = BeautifulSoup(resp.text, 'html.parser') datasets = [] for item in soup.select('.dataset-item'): dataset = { 'title': item.select_one('.title').text.strip(), 'format': item.select_one('.format').text.strip(), 'dept': item.select_one('.dept').text.strip(), 'update': item.select_one('.date').text.strip(), } datasets.append(dataset) return datasets
python # 动态页面采集：北京市政务数据资源网（Vue渲染） from selenium import webdriver from selenium.webdriver.chrome.options import Options class DynamicCollector: def __init__(self): options = Options() options.add_argument('--headless') options.add_argument('--no-sandbox') self.driver = webdriver.Chrome(options=options) def collect(self, url): self.driver.get(url) WebDriverWait(self.driver, 10).until( EC.presence_of_element_located((By.CSS_SELECTOR, '.dataset-list')) ) for _ in range(5): self.driver.execute_script('window.scrollTo(0, document.body.scrollHeight);') time.sleep(1) html = self.driver.page_source # 后续解析与静态采集相同...
python # API接口采集：贵州省数据开放平台 class APICollector: def collect(self, api_endpoint): params = {'page': 1, 'pageSize': 100} all_data = [] while True: resp = requests.get(api_endpoint, params=params, timeout=30) result = resp.json() all_data.extend(result['data']['list']) if result['data']['pageNum'] >= result['data']['pages']: break params['page'] += 1 return all_data
