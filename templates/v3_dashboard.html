系统技术架构
采集引擎
• 动态请求：Selenium + ChromeDriver
• 静态解析：Requests + BeautifulSoup4
• 反爬策略：随机UA + 代理池 + 频率控制
• 容错机制：自动重试 + 异常捕获 + 断点续采
数据治理
• 存储层：SQLite + JSON + CSV
• 清洗流程：去重 → 标准化 → 校验 → 关联
• 版本管理：Git + 数据库快照
• 增量更新：哈希比对 + 时间戳追踪
分析引擎
• 排名分析：TOPSIS（逼近理想解）
• 效率评估：DEA（数据包络分析）
• 因果挖掘：DEMATEL（决策试验法）
• 路径分析：fsQCA（模糊集定性比较）
• 政策评估：DID（双重差分法）
展示层
• 后端框架：Flask + Jinja2 模板
• 前端图表：ECharts 5.x + D3.js 7.x
• 响应式布局：CSS Grid + Flexbox
• 部署环境：腾讯云 + Nginx + Gunicorn
