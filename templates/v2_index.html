三层架构采集系统

数据采集层

覆盖 88 个政府数据开放平台，省级/副省级/地市级三层架构，支持 HTTP/HTTPS 探测、HTML 解析、API 接口调用、批量下载检测。

处理存储层

SQLite 关系型数据库 + JSON 半结构化存储，采集日志自动归档，数据血缘追踪，质量评分自动计算，支持增量更新与版本快照。

分析展示层

Flask Web 服务 + Bootstrap 响应式前端 + Chart.js 交互式图表，支持实时数据看板、4E 维度切换、评估结果导出、论文图表生成。
